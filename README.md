# Панова Анастасия БАС-2 Лабораторная №4

# Задание 1

## Задача 1

### Текст задачи

Дерево содержит целые числа. Вычесть из каждого разряда числа 1, если разряд не содержит 0, иначе заменить этот разряд на 1.

### Алгоритм решения

Программа заполняет дерево десятью случайными числами из диапаона от 0 до 1000 и выводит его на экран. Заполнение происходит по правилам бинарого дерева поиска. С помощью функции map каждый узел дерева проходит через функцию обработки. Эта функция понижает все разряды элемента (числа узла) на 1 или заменяет разряд со значением 0 на 1. В момент обработки элемент из целого числа превращается в список его рарядов, а после снова собирается в число. Новое дерево выводится на экран.

# Задание 2

## Задача 1

### Текст задачи

Дерево содержит строки. Сколько узлов дерева заканчивается на заданный символ?

### Алгоритм решения

Программа создаёт бинарное дерево, которое заполняется пользователем. Пользователь вводит строки, которые вносятся в дерево в качестве его элементов (узлов). С помощью функции fold происходит обход дерева. Программа проверяет поседний символ в строке элемента на соответствие символу, введённому пользователем, проходя так всё дерево. Подходящие узлы подсчитываются, полученное число выводиться на экран.
