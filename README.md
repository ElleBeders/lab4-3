# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего римские обозначения цифр от I до IX, получить их десятичные представления.

### Алгоритм решения

Программа принимает от пользователя строку с римскими цифрами, затем с помощью словаря наход соответствия между римскими и десятичными цифрами. Если соответствия нет, вписывается 0, иначе - цифра из словаря. Результат выводиться на экран.

### Тестирование

Скриншоты результата работы программы

## Задача 2

### Текст задачи

Список содержит десятичные цифры. Составить число из чётных цифр.

### Алгоритм решения

Пользователь вводит строку из цифр, программа отфильтровывае нечётные цифры, проверяет, не стала ли последовательность пустой. Если стала - выводиться 0, иначе составляется число из чётных цифр, которые остались после сортировки. Число выводится на экран.

### Тестирование

Скриншоты результата работы программы
